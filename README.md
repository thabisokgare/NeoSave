


# 💸 NeoSave — Smart AI-Powered Budgeting & Savings App

NeoSave is a modern, AI-powered personal finance app designed to help users take control of their money. With intelligent budgeting, automated savings, and personalized investment guidance, NeoSave is your all-in-one platform to achieve financial wellness.

---

## ✨ Features

- 🔐 **Secure Sign In** with Google or email
- 📊 **Interactive Survey** after sign-up to understand income, spending habits, and savings goals
- 🧠 **AI-Generated Budgets & Savings Plans** based on user input and bank card transactions
- 💡 **Financial Insights** personalized with AI
- 🎯 **Savings Goal Tracker**
- 📚 **Gamified Investment Learning** with progress tracking
- 💳 **Bank Spending Analysis** to suggest improved plans
- 🔄 **Dynamic Budgets** updated as user spending changes
- 🌐 **Responsive Web & Mobile-Friendly UI** (TailwindCSS, React)
- 🎞️ **GSAP Animations** for smooth, engaging UI transitions

## 🚀 Getting Started

### Prerequisites
- Node.js & npm
- .NET 7 SDK or later
- SQL Server (LocalDB or full)
- VS Code or JetBrains Rider

### Clone the Repo
```bash
git clone https://github.com/thabisokgare/NeoSave.git
cd NeoSave
````

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd NeoSave.API
dotnet ef database update
dotnet run
```
